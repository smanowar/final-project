Stackoverflow Response Time Predictor

How long will it take for my question to receive an accepted answer on Stack Overflow?

UofTSCS Data Analytis Bootcamp 2021 Final Project Submission

About Our Project

Time is valuable; and we don’t want to sit around waiting for answers. As programming students, we use online resources to ask questions to build our knowledge, troubleshoot code and expand our problem solving skills. But, once we ask that question, we have no idea when we can expect a response.

Should we stick around because answers will start coming in within the hour?
Or step away from the problem and come back tomorrow for an answer?
Or keep researching and combing through documentation because it’s going to be a while?
So we set out to answer the question:

How long will it take for my question to receive an accepted answer on Stack Overflow?

Making a Prediction: Machine Learning

How it Works:

The predictor uses a Random Forest Classifier to make a prediction. Random Forest is a type of Machine Learning in which a computer is fed an algorithm based on previous outcomes to make an educated guess when fed new information. Our model was trained using Stackoverflow messageboard data taken from Kaggle for the date range January 2021 to September 2021.


Results:




Our predictor makes predictions with an accuracy of:


            55%


This means that when you get an answer from our model, the answer will be correct 55% of the time








How Do The Inputs Affect the Outcome?



The inputs in machine learning are referred to as features. Feature importance refers to techniques that assign a score to input features based on how useful they are at predicting a target variable. The higher the score, the greater influence it has on the prediction.
